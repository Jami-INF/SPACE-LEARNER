Application destinée à l'apprentissage de l'espace

Les fonctionnalités sont : 
- Participer a un quiz de dix questions
- Voir une anecdote sur l'espace au hasard
- Afficher sur demande un astre au hasard pour en apprendre plus sur celui-ci
- Ajouter un astre ainsi que sa photo a la liste d'astres
- Consulter la liste d'astre et afficher ceux d'un type à la fois grace a une combobox
- Pouvoir modifier et supprimer un astre
- Sauvegarder les données dans des fichiers XML

La documentation complète du projet réalisé en première année d'IUT Informatique 2020-2021 :

https://docs.google.com/presentation/d/1hS0k_OdIGS3CynoUPR-jOP1YhJKIRRD3pAr_vXs8czs/edit?usp=sharing

Les données de l'application actuelle sont absolument factices, elles sont présentes uniquement pour tester le code.
